### **第七章：无线网络与移动网络**

**📌 从Wi-Fi到5G：无线连接的演进图谱**

------

#### 1. 无线局域网 WLAN

**📊 802.11标准家族**

| **标准** | **频段** | **数据率** | **技术**  |
| -------- | -------- | ---------- | --------- |
| 802.11b  | 2.4GHz   | 11 Mbps    | DSSS      |
| 802.11n  | 2.4/5GHz | 600 Mbps   | OFDM+MIMO |

**🛠 CSMA/CA退避算法**
检测冲突 → 随机退避时间 → 重试发送 ✅

------

#### 2. 蜂窝移动网络演进

**📌 4G LTE核心特性**

- **IP架构** ：全IP化支持多媒体
- **高数据率** ：100 Mbps+
- **低延迟** ：VoLTE语音优化

**🔵 移动IP三角形路由问题**
归属代理转发 → 直接路由优化 ✅

------

#### 3. 无线网络挑战

**📌 TCP在无线环境适配**

- **本地恢复** ：链路层重传避免TCP降速
- **拆分连接** ：无线段与有线段分离处理
